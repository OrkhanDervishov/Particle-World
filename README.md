<h2>BUTTONS</h2>

1. Mouse left - Create partciles
2. Mouse right - Delete partciles
3. Screen clear - C
4. Particle type change - V
5. Explosion effect - R
6. Heatmap ON/OFF - F
7. Brush radius increase - 1
8. Brush radius decrease - 2
9. Delay increase - 3
10. Delay decrease - 4
11. Quit - Esc
